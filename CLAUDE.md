# QuickShop

Mini plataforma de e-commerce con catalogo de productos, carrito, pedidos y gestion de usuarios por roles (cliente y administrador).

Stack: FastAPI + SQLAlchemy 2.0 + PostgreSQL 15 en el backend; React 18 + Vite + Tailwind CSS en el frontend. Despliegue con Docker Compose.

Para convenciones generales, estructura de carpetas, orden de desarrollo y configuracion de entorno consulta @AGENTS.md.

Para instrucciones especificas del backend consulta @backend/AGENTS.md.

Para instrucciones especificas del frontend consulta @frontend/AGENTS.md.

Para las especificaciones detalladas de cada modulo consulta los archivos en docs/specs-prds/.
