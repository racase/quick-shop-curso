# quick-shop-curso
Repositorio para el curso Desarrollo con IA: De escribir prompts al desarrollo con agentes

El repositorio no tiene codigo, el codigo se irá generando durante el curso con agentes de codigo, en especial Claude Code y OpenCode. 

La rama main solo tiene las carpetas iniciales necesarias a modo de scaffolding para poder clonar el proyecto y empezar a trabajar en el con los principales agentes de codigo.

## 🤖 Cómo trabajar con los Agentes en este Curso
Durante las sesiones prácticas utilizaremos las configuraciones de este repositorio para dar contexto a los asistentes:
*   **Reglas de estilo y desarrollo**: Cada carpeta de agente puede contener archivos de instrucciones de comportamiento (por ejemplo, `.claude` o archivos de configuración personalizados) para indicarle al agente cómo estructurar el código del e-commerce "Quick Shop".
*   **Contexto de negocio**: Encontrarás guías rápidas en `docs/` que los agentes pueden leer para entender las reglas de negocio de la aplicación antes de programar.
