# frontend/CLAUDE.md

Instrucciones especificas del frontend de QuickShop (React 18 + Vite + Tailwind CSS v4 + React Router v6).

Los detalles completos de convenciones, estructura, rutas, configuracion de pnpm 11 y el Dockerfile estan en @AGENTS.md.

El sistema de diseno visual (colores, tipografia, botones, elevaciones, las dos pistas cinematica/transaccional) esta definido en @DESIGN.md. Cualquier decision de UI debe seguir ese fichero. Las reglas criticas y el mapeo a clases Tailwind estan resumidos en la seccion "Sistema de diseno" de @AGENTS.md.

Las especificaciones de negocio (pantallas requeridas por modulo) estan en `../docs/specs-prds/`.
