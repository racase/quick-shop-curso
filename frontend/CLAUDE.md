# Frontend — QuickShop

SPA construida con React 18 + Vite + Tailwind CSS + React Router. Gestionada con pnpm 11.

## Arranque rapido (desarrollo local)

```bash
cd frontend
cp .env.example .env
pnpm install
pnpm dev
```
