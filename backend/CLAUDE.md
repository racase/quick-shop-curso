# Backend - QuickShop

API REST construida con FastAPI, SQLAlchemy 2.0 (async) y PostgreSQL 15. Migraciones con Alembic, gestion de dependencias con uv, autenticacion JWT con bcrypt.

Para convenciones de codigo, patrones de FastAPI, configuracion de base de datos, guia de migraciones y politica de autenticacion consulta @AGENTS.md.

## Regla de Idioma del Agente
- **El agente debe comunicarse e interactuar siempre en castellano (español).**
